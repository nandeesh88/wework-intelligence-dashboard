# WeWork Occupancy & Revenue Intelligence Dashboard

## Executive Summary

**Project Overview**  
This project is an AI-powered business intelligence dashboard designed to analyze
workspace occupancy, member utilization, and revenue trends across multiple
WeWork locations. The solution demonstrates end-to-end BI capabilities, from
data generation and validation to interactive visualization and executive reporting.

**Key Features**
- Occupancy and utilization analysis across 5 locations
- Revenue trend analysis and performance metrics
- Interactive dashboards with filters and charts
- AI-ready insights module for business recommendations
- Automated data generation and data quality validation

**Technical Skills Demonstrated**
- Python scripting for data generation and quality checks
- Data analysis using Pandas and NumPy
- Interactive dashboard development using React and Recharts
- Business intelligence reporting and KPI analysis
- Version control and deployment using GitHub Pages

**Business Insights**
- Identified peak usage hours between 11 AM and 3 PM
- Average occupancy rate of approximately 80% across locations
- Revenue trends highlighting high-performing locations
- Actionable insights for space optimization and member retention

**Technology Stack**
Python, Pandas, NumPy, React, JavaScript, Recharts, GitHub Pages
